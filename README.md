# 重返未来1999抽数计算器

一个用于计算《重返未来1999》从指定日期开始到截止日期可获得的纯雨滴、粹雨滴、独一律数量及对应抽数的网页工具。

## 功能特性

### 日常资源管理
- **纯雨滴**：每月签到、每日任务、小月卡、每周任务、深眠域（每月1号）、醒梦域（每月16号）
- **粹雨滴**：小月卡（购买即送）
- **独一律**：迷途之声商店（每月）、免费档大月卡、付费档大月卡

### 限时活动管理
- 支持添加自定义限时活动
- 内置限时活动（赠送、签到、剧情活动、维护补偿、版本预热）
- 支持稳定/不稳定类型活动
- 支持单次/每日频率活动

### 大容量迷途之声商店
- 自定义输入大容量迷途之声数量
- 自动计算可兑换的独一律数量
- 兑换比例：10大容量迷途之声 = 1独一律

### 其他功能
- **粹雨滴不用于抽卡**：可选排除粹雨滴参与抽数计算
- **日期快捷键**：快速设置日期
- **主题切换**：支持深色/浅色模式
- **最低/平均/最高抽数计算**：考虑不同获取可能

## 使用方法

### 本地运行

1. 进入项目目录
```bash
cd reverse1999_calculator
```

2. 启动本地服务器
```bash
python -m http.server 8080
```

3. 访问 `http://localhost:8080/index.html`

### 线上运行

`https://xenon-xys.github.io/reverse1999_calculator/`

## 资源折算比例

| 资源 | 折算比例 |
|------|----------|
| 纯雨滴 | 180 = 1抽 |
| 粹雨滴 | 180 = 1抽 |
| 独一律 | 1 = 1抽 |
| 大容量迷途之声 | 10 = 1独一律 |

## 游戏时间规则

游戏时间划分：**每天凌晨5:00 至 次日凌晨4:59:59**

在凌晨5:00之前的时间属于前一天的游戏时间。
